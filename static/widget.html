Setup

Running in 5 minutes. Seriously.

No complex infrastructure. No Docker required. Just Python, an API key, and you're live.

Install dependencies

pip install -r requirements.txt

Python 3.9+ and about 30 seconds of patience.

Add your API key

Copy .env.example to .env, paste your OpenAI key. That's the whole config.

Start the server

python main.py

Opens on port 8400. Knowledge base auto-ingests on startup.

Chat with CloverBot

Open localhost:8400/widget and start a conversation. When you're ready, embed on your site with one <script> tag.

<!-- Add this before </body> on any page --> <script src="https://your-server:8400/static/cloverbot.js" data-api-url="https://your-server:8400"> </script>
